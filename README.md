# 2Square Admin Panel

Admin dashboard for managing the 2Square e-commerce platform.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3001](http://localhost:3001) in your browser.

## Features

- Dashboard with analytics
- Product management
- User management
- Order management
- Settings
